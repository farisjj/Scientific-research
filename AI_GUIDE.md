# دليل تطوير موقع البحث العلمي الطبي (MedRes) 🚀

هذا الملف مخصص لمساعدة المطورين أو أنظمة الذكاء الاصطناعي على فهم وبناء وتعديل هذا المشروع بسرعة وكفاءة.

## 📌 نظرة عامة على المشروع
الموقع عبارة عن دليل إرشادي شامل للبحث العلمي الطبي، مبني باستخدام تقنيات حديثة لضمان السرعة والتجاوب الكامل.

- **التقنيات المستخدمة:** React, TypeScript, Vite, Tailwind CSS, Lucide React (الأيقونات).
- **التصميم:** ثيم مظلم (Dark Theme) بألوان Navy و Neon Blue.
- **اللغة:** يدعم اللغة العربية بشكل كامل (RTL).

## 📂 هيكلية المجلدات
- `client/src/components/`: يحتوي على المكونات القابلة لإعادة الاستخدام (مثل هرم الأدلة، شريط البحث، التنقل).
- `client/src/pages/`: يحتوي على الصفحات والأقسام الرئيسية للموقع.
- `client/public/`: يحتوي على الصور والأصول الثابتة (بما في ذلك الصور التي تم توليدها بالذكاء الاصطناعي).
- `client/src/index.css`: يحتوي على التنسيقات العامة والـ Custom Classes (مثل `.section-hero-image`).

## 🛠 كيفية التشغيل (Build & Run)
1. **تثبيت الاعتمادات:**
   ```bash
   cd client
   pnpm install
   ```
2. **تشغيل خادم التطوير:**
   ```bash
   pnpm run dev
   ```
3. **بناء النسخة النهائية:**
   ```bash
   pnpm run build
   ```

## 💡 ملاحظات هامة للذكاء الاصطناعي (AI Instructions)
- **التفاعلية:** مكون `EvidencePyramid.tsx` يدعم التمرير (Hover) على الحاسوب والنقر (Click) على الموبايل. لا تقم بإضافة `touch-none` لأنها تمنع التمرير.
- **الصور:** جميع الصور في الأقسام الأخيرة تستخدم كلاس `.section-hero-image` لضمان التناسق والتوهج الأزرق.
- **RTL:** المشروع مصمم ليكون `dir="rtl"`. تأكد من الحفاظ على محاذاة النصوص والاتجاهات عند إضافة مكونات جديدة.
- **التجاوب:** استخدم Tailwind classes (مثل `sm:`, `md:`, `lg:`) لضمان عمل الواجهة بشكل ممتاز على الهواتف.

- **Primary/Accent:** Blue 500 / Cyan 400 (Neon Blue)
